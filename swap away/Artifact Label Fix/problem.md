## Title

Artifact Label Fix

## Slug

artifact-label-fix

## Difficulty

Easy

## Description

Two artifact labels in the museum database are almost identical.  
Determine if the first label can be made equal to the second by swapping exactly two characters.

If yes, print `"YES"`; otherwise, print `"NO"`.

## Examples

### 1

#### Input

5
abcde
abced

#### Output
YES

### 2

#### Input

6
xyzzaz
qwdsad

#### Output
NO

## Input Format  

- Line 1: integer `n`.  
- Line 2: first label.  
- Line 3: second label.

## Output Format  

- `"YES"` or `"NO"`.

## Constraints  

- 1 ≤ n ≤ 10⁵  
- Lowercase English letters only.  

## Time Limit

1 second

## Memory Limit

512 MB

## Tags

strings, swapping.
