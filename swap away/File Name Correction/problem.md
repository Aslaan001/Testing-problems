## Title

File Name Correction

## Slug

file-name-correction

## Difficulty

Easy

## Description

Two versions of a file name differ by only a small mistake.  
Check if the first file name can become identical to the second by swapping exactly two letters.

Return `"YES"` if possible; otherwise, return `"NO"`.

## Examples

### 1

#### Input

5
abcde
abced

#### Output
YES

### 2

#### Input

6
xyzzaz
qwdsad

#### Output
NO

## Input Format  

- Line 1: integer `n`.  
- Line 2: first name.  
- Line 3: second name.

## Output Format  

- `"YES"` or `"NO"`.

## Constraints  

- 1 ≤ n ≤ 10⁵  
- Lowercase English letters only.  

## Time Limit

1 second

## Memory Limit

512 MB

## Tags

strings, swapping.
