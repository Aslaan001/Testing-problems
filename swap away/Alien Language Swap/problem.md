## Title

Alien Language Swap

## Slug

alien-language-swap

## Difficulty

Easy

## Description

Two messages from an alien language have been received.  
You need to determine if one can be transformed into the other by swapping exactly two letters in the first message.

Return `"YES"` if this transformation is possible, otherwise `"NO"`.

## Examples

### 1

#### Input

5
abcde
abced

#### Output
YES

### 2

#### Input

6
xyzzaz
qwdsad

#### Output
NO

## Input Format  

- First line: integer `n`.  
- Second and third lines: the two alien strings.

## Output Format  

- `"YES"` or `"NO"`.

## Constraints  

- 1 ≤ n ≤ 10⁵  
- Lowercase English letters only.  

## Time Limit

1 second

## Memory Limit

512 MB

## Tags

strings, swapping.
