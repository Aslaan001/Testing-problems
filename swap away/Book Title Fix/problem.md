## Title

Book Title Fix

## Slug

book-title-fix

## Difficulty

Easy

## Description

Two copies of a book title have a minor printing error.  
Check if the first title can be made identical to the second by swapping exactly two letters.

Return `"YES"` if possible; otherwise, return `"NO"`.

## Examples

### 1

#### Input

5
abcde
abced

#### Output
YES

### 2

#### Input

6
xyzzaz
qwdsad

#### Output
NO

## Input Format  

- Line 1: integer `n`.  
- Line 2: title 1.  
- Line 3: title 2.

## Output Format  

- `"YES"` or `"NO"`.

## Constraints  

- 1 ≤ n ≤ 10⁵  
- Lowercase English letters only.  

## Time Limit

1 second

## Memory Limit

512 MB

## Tags

strings, swapping.
