## Title

Word Anagram Swap

## Slug

word-anagram-swap

## Difficulty

Easy

## Description

You are comparing two words to see if they can become identical after swapping exactly two letters in the first word.

If one swap can make them equal, print `"YES"`. Otherwise, print `"NO"`.

## Examples

### 1

#### Input

5
abcde
abced

#### Output
YES

### 2

#### Input

6
xyzzaz
qwdsad

#### Output
NO

## Input Format  

- First line: integer `n`.  
- Second and third lines: the two words.

## Output Format  

- `"YES"` or `"NO"`.

## Constraints  

- 1 ≤ n ≤ 10⁵  
- Lowercase English letters only.  

## Time Limit

1 second

## Memory Limit

512 MB

## Tags

strings, swapping.
