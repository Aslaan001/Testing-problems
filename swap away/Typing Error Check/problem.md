## Title

Typing Error Check

## Slug

typing-error-check

## Difficulty

Easy

## Description

A typist made a mistake while entering a word.  
Determine whether swapping exactly two letters in the first version can make it identical to the correct version.

Return `"YES"` if one swap fixes it, otherwise `"NO"`.

## Examples

### 1

#### Input

5
abcde
abced

#### Output
YES

### 2

#### Input

6
xyzzaz
qwdsad

#### Output
NO

## Input Format  

- Line 1: integer `n`.  
- Line 2: first typed word.  
- Line 3: correct word.

## Output Format  

- `"YES"` or `"NO"`.

## Constraints  

- 1 ≤ n ≤ 10⁵  
- Lowercase English letters only.  

## Time Limit

1 second

## Memory Limit

512 MB

## Tags

strings, swapping.
