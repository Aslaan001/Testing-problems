## Title
Equilibrium Stones of the Mountain Keepers

## Slug
equilibrium-stones-of-the-mountain-keepers

## Difficulty
Hard

## Description

In this reimagined setting, equilibrium stones of the mountain keepers define a special class of numbers. For any value within the given interval, its digit sum and digit product determine whether it carries a mark of harmony. A number is considered beautiful if its product is perfectly divisible by its sum, maintaining numerical balance. Your task is to determine how many such harmonized values exist within the specified range.

## Examples

### 1

#### Input
10 20

#### Output
2

### 2

#### Input
1 15

#### Output
10

## Input Format

Two space-separated integers l and r.

## Output Format

Return a single integer — the number of beautiful numbers in the range [l, r].

## Constraints

1 ≤ l ≤ r < 1000000000

## Time Limit
1 second

## Memory Limit
512 MB
