## Title

Reorder Smallest File Front

## Slug

reorder-smallest-file-front

## Difficulty

Easy

## Description

A system stores file sizes in an array.  
Move the `smallest file` to the front of the list by swapping it with the first file.  
If it’s already first, keep the order unchanged.

## Examples

### 1

#### Input

5
5 3 4 2 1 

#### Output
1 3 4 2 5

### 2

#### Input

4
2 1 3 4 

#### Output
1 2 3 4

## Input Format  

- Line 1: integer `n`.  
- Line 2: file sizes.

## Output Format  

- The updated list after swap.

## Constraints  

- 1 ≤ n ≤ 10⁴  
- -10⁹ ≤ arr[i] ≤ 10⁹  

## Time Limit

1 second

## Memory Limit

512 MB

## Tags

arrays, swapping.
