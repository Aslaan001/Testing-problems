## Title

Symmetry Index

## Slug

symmetry-index

## Difficulty

Easy

## Description

In a symmetric structure represented as an array, you need to find the smallest index that divides the structure into two equal halves by sum.  

If symmetry cannot be achieved, return `-1`.

## Examples

### 1

#### Input

5
1 7 3 6 5 6 

#### Output
3

### 2

#### Input

4  
2 2 2 2 

#### Output
-1

## Input Format  

- First line: integer `n`.  
- Second line: `n` integers — structure values.

## Output Format  

- Integer — symmetry index or `-1`.

## Constraints  

- 1 ≤ n ≤ 10⁴  
- -10⁹ ≤ arr[i] ≤ 10⁹    

## Time Limit

1 second

## Memory Limit

512 MB
