## Title

Sorted or Not

## Slug

sorted-or-not

## Difficulty

Easy

## Description

You need to verify if an array `arr[]` of `n` elements is `sorted in non-decreasing order`.

Return `YES` if sorted, otherwise return `NO`.

---

### Example 1

#### Input
5  
1 2 3 4 5

#### Output
YES

### Example 2

#### Input
5  
5 4 3 2 1

#### Output
NO

## Input Format
- First line: integer `n`  
- Second line: `n` integers

## Output Format
`YES` or `NO`

## Constraints
1 ≤ n ≤ 10⁶  
1 ≤ arr[i] ≤ 10⁶  

## Tags
arrays, sorting
