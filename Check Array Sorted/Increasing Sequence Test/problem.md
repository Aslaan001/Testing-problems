## Title

Increasing Sequence Test

## Slug

increasing-sequence-test

## Difficulty

Easy

## Description

Given an array `arr[]`, determine if it forms an `increasing or flat (non-decreasing)` sequence.

Print `YES` if sorted, otherwise print `NO`.

---

### Example 1

#### Input
4  
2 4 4 9

#### Output
YES

### Example 2

#### Input
4  
2 4 3 9

#### Output
NO

## Input Format
- First line: integer `n`  
- Second line: `n` integers

## Output Format
`YES` or `NO`

## Constraints
1 ≤ n ≤ 10⁶  
1 ≤ arr[i] ≤ 10⁶  

## Tags
arrays, sorting
