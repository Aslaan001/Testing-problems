## Title

Verify Ordering

## Slug

verify-ordering

## Difficulty

Easy

## Description

Given an array of `n` integers, verify whether it is sorted in `non-decreasing order`.


### Example 1
#### Input
5  
1 1 3 5 8

#### Output
YES

### Example 2
#### Input
5  
1 3 2 4 5

#### Output
NO

## Input Format
n  
array elements

## Output Format
YES or NO

## Constraints
1 ≤ n ≤ 10⁶  
1 ≤ arr[i] ≤ 10⁶  

## Tags
arrays, sorting
