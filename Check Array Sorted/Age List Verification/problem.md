## Title
Age List Verification

## Slug
age-list-verification

## Difficulty
Easy

## Description
An event’s guest list `arr[]` contains ages sorted by check-in order. Confirm that no younger guest appears after an older one. Return `true` if the list is in `non-decreasing order`, else `false`.

### Example 1
#### Input
4  
21 21 25 30
#### Output
true

### Example 2
#### Input
4  
21 25 23 30
#### Output
false

## Input Format
`n`  
`arr[]` guest ages

## Output Format
`true` or `false`

## Constraints
1 ≤ n ≤ 10⁶  
1 ≤ arr[i] ≤ 10⁶

## Tags
arrays, sorting
