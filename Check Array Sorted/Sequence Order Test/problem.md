## Title

Sequence Order Test

## Slug

sequence-order-test

## Difficulty

Easy

## Description

Given an integer array `arr[]`, determine if it is sorted in `non-decreasing order`.

If sorted, print `YES`; otherwise, print `NO`.

---

### Example 1
#### Input
4  
1 2 2 3

#### Output
YES

### Example 2
#### Input
4  
3 2 2 1

#### Output
NO

## Input Format
n  
array elements

## Output Format
`YES` or `NO`

## Constraints
1 ≤ n ≤ 10⁶  
1 ≤ arr[i] ≤ 10⁶  

## Tags
arrays, sorting
