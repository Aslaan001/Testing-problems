## Title
Exam Score Checker

## Slug
exam-score-checker

## Difficulty
Easy

## Description
You are given an array `arr[]` representing a student’s scores over multiple tests. Determine whether their performance is consistent or improving. Return `true` if scores are in `non-decreasing order`, otherwise `false`.

### Example 1
#### Input
5  
50 55 55 60 70
#### Output
true

### Example 2
#### Input
5  
80 75 85 90 95
#### Output
false

## Input Format
`n`  
`arr[]` test scores

## Output Format
`true` or `false`

## Constraints
1 ≤ n ≤ 10⁶  
1 ≤ arr[i] ≤ 10⁶

## Tags
arrays, sorting
