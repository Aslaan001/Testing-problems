## Title
Movie Seat Rows

## Slug
movie-seat-rows

## Difficulty
Easy

## Description
Tickets are scanned in order of row numbers stored in `arr[]`. To keep aisle flow, row numbers should never decrease. Check whether `arr[]` is in `non-decreasing order`. Return `true` if sorted, else `false`.

### Example 1
#### Input
5  
3 3 4 5 5
#### Output
true

### Example 2
#### Input
5  
3 5 4 6 6
#### Output
false

## Input Format
- `n`  
- `arr[]` row numbers

## Output Format
`true` or `false`

## Constraints
1 ≤ n ≤ 10⁶  
1 ≤ arr[i] ≤ 10⁶

## Tags
arrays, sorting
