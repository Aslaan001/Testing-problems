## Title

Array In Order

## Slug

array-in-order

## Difficulty

Easy

## Description

Given `n` integers, check whether the sequence is in `non-decreasing order`.

Output `YES` if it is sorted, else `NO`.


### Example 1
#### Input
5  
1 2 2 2 3

#### Output
YES

### Example 2
#### Input
4  
4 3 2 1

#### Output
NO

## Input Format
- `n`  
- Array elements

## Output Format
`YES` or `NO`

## Constraints
1 ≤ n ≤ 10⁶  
1 ≤ arr[i] ≤ 10⁶  

## Tags
arrays, sorting
