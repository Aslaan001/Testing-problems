## Title

Increasing Validation

## Slug

increasing-validation

## Difficulty

Easy

## Description

You are provided an integer array `arr[]`.  
Check if it is in `non-decreasing order`.  
If the order is correct, print `YES`; otherwise, print `NO`.

---

### Example 1
#### Input
5  
2 3 4 5 6

#### Output
YES

### Example 2
#### Input
5  
2 5 4 6 7

#### Output
NO

## Input Format
`n`  
`arr[]`

## Output Format
`YES` or `NO`

## Constraints
1 ≤ n ≤ 10⁶  
1 ≤ arr[i] ≤ 10⁶  

## Tags
arrays, sorting
