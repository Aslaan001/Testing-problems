## Title
Robot Speed Calibration

## Slug
robot-speed-calibration

## Difficulty
Easy

## Description
During calibration, a robot ramps speed in steps recorded as `arr[]`. Verify there is no slowdown between steps. Check if `arr[]` is in `non-decreasing order`. Output `true` if correct, else `false`.

### Example 1
#### Input
5  
1 2 2 3 3
#### Output
true

### Example 2
#### Input
5  
1 3 2 4 5
#### Output
false

## Input Format
- `n` — number of steps  
- `arr[]` — speeds

## Output Format
`true` or `false`

## Constraints
1 ≤ n ≤ 10⁶  
1 ≤ arr[i] ≤ 10⁶

## Tags
arrays, sorting
