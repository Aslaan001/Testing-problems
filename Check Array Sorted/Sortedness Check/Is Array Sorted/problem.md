## Title

Is Array Sorted?

## Slug

is-array-sorted

## Difficulty

Easy

## Description

You are given a list of integers `arr[]`.  
Check if the array is sorted in non-decreasing order.

Print `YES` if sorted, else `NO`.


### Example 1

#### Input
6  
1 1 2 3 3 4

#### Output
YES

### Example 2

#### Input
3  
3 2 1

#### Output
NO

## Input Format

- First line: integer `n`  
- Second line: `n` integers

## Output Format

- `"YES"` or `"NO"`

## Constraints

1 ≤ n ≤ 10⁶  
1 ≤ arr[i] ≤ 10⁶

## Tags
arrays, sorting
