## Title

Validate Sequence

## Slug

validate-sequence

## Difficulty

Easy

## Description

You are given an array `arr[]`.  
Check if it is sorted in `non-decreasing order`.  
If true, output `YES`; otherwise, output `NO`.

---

### Example 1

#### Input
5  
1 1 1 2 3

#### Output
YES

### Example 2

#### Input
5  
2 1 2 3 4

#### Output
NO

## Input Format
- Integer `n`  
- Array elements

## Output Format
`YES` or `NO`

## Constraints
1 ≤ n ≤ 10⁶  
1 ≤ arr[i] ≤ 10⁶  

## Tags
arrays, sorting
