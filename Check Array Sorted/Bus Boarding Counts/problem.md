## Title
Bus Boarding Counts

## Slug
bus-boarding-counts

## Difficulty
Easy

## Description
At consecutive stops, a counter logs total passengers on board as `arr[]`. Since passengers only board (for this route), totals should be non-decreasing. Verify if `arr[]` is in `non-decreasing order`. Output `true` or `false`.

### Example 1
#### Input
5  
5 5 7 8 8
#### Output
true

### Example 2
#### Input
5  
5 6 5 7 9
#### Output
false

## Input Format
- `n`  
- `arr[]` totals

## Output Format
`true` or `false`

## Constraints
1 ≤ n ≤ 10⁶  
1 ≤ arr[i] ≤ 10⁶

## Tags
arrays, sorting
