## Title

Non-Decreasing Test

## Slug

nondecreasing-test

## Difficulty

Easy

## Description

You are given an array `arr[]`. Your job is to check if the array elements are sorted in `non-decreasing order`.

If yes, print `YES`; else, `NO`.


### Example 1
#### Input
4  
2 2 3 5

#### Output
YES

### Example 2
#### Input
4  
2 3 2 5

#### Output
NO

## Input Format
- `n`  
- Array of integers

## Output Format
- `"YES"` or `"NO"`

## Constraints
1 ≤ n ≤ 10⁶  
1 ≤ arr[i] ≤ 10⁶  

## Tags
arrays, sorting
