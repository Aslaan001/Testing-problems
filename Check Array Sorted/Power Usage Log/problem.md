## Title
Power Usage Log

## Slug
power-usage-log

## Difficulty
Easy

## Description
A smart meter records hourly power usage in `arr[]`. To validate a rising-demand hypothesis, check whether readings never decrease. Determine if `arr[]` is in `non-decreasing order`. Output `true` if sorted, else `false`.

### Example 1
#### Input
6  
2 2 3 3 3 4
#### Output
true

### Example 2
#### Input
6  
2 3 2 4 4 5
#### Output
false

## Input Format
- `n` — number of hours  
- `arr[]` — usage values

## Output Format
`true` or `false`

## Constraints
1 ≤ n ≤ 10⁶  
1 ≤ arr[i] ≤ 10⁶

## Tags
arrays, sorting
