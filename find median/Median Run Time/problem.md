## Title

Median Run Time

## Slug

median-run-time

## Difficulty

Easy

## Description

Given run times (in seconds) for multiple tasks, sort them and return the median run time. For even counts, average two middle values using integer division.

### Example 1

#### Input

5
2 5 1 7 3

#### Output
3

### 2

#### Input
6
1 2 3 4 5 6

#### Output
3

## Input Format

- First line: `n`  
- Second line: `n` integers — run times.

## Output Format

- Single integer — median run time.

## Constraints

- 1 ≤ n ≤ 10⁶  
- 1 ≤ arr[i] ≤ 10⁶

## Time Limit

1 second

## Memory Limit

512 MB

## Tags

arrays, sorting.
