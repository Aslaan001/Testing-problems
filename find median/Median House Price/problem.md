## Title

Median House Price

## Slug

median-house-price

## Difficulty

Easy

## Description

You are given house prices as integers in an array. Sort prices and return the median price. For even number of houses, return integer average of two middle prices.

### Example 1

#### Input

5
2 5 1 7 3

#### Output
3

### 2

#### Input
6
1 2 3 4 5 6

#### Output
3

## Input Format

- First line: `n`  
- Second line: `n` integers — prices.

## Output Format

- Single integer — median price.

## Constraints

- 1 ≤ n ≤ 10⁶  
- 1 ≤ arr[i] ≤ 10⁶

## Time Limit

1 second

## Memory Limit

512 MB

## Tags

arrays, sorting.
