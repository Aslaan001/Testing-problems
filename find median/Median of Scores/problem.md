## Title

Median of Scores

## Slug

median-of-scores

## Difficulty

Easy

## Description

Given an array of integer scores, sort and compute the median. If even length, return integer division of middle two.

### Example 1

#### Input

5
2 5 1 7 3

#### Output
3

### 2

#### Input
6
1 2 3 4 5 6

#### Output
3

## Input Format

- First line: `n`  
- Second line: `n` integers — scores.

## Output Format

- Single integer — median score.

## Constraints

- 1 ≤ n ≤ 10⁶  
- 1 ≤ arr[i] ≤ 10⁶

## Time Limit

1 second

## Memory Limit

512 MB

## Tags

arrays, sorting.
