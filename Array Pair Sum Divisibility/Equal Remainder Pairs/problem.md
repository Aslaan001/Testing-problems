## Title

Equal Remainder Pairs

## Slug

equal-remainder-pairs

## Difficulty

Medium

## Description

Given an array `arr[]` and a number `k`, check whether you can pair elements so that each pair has the same remainder when divided by `k`.

If possible, print `"YES"`. Otherwise, print `"NO"`.

### Example 1

#### Input
4  
9 7 5 3  
6

#### Output
YES

### Example 2

#### Input
4  
92 75 65 48  
10

#### Output
YES

## Input Format

- First line: integer `n`.  
- Second line: array elements.  
- Third line: integer `k`.

## Output Format

- `"YES"` or `"NO"`.

## Constraints

- 1 ≤ n ≤ 10⁵  
- 1 ≤ arr[i] ≤ 10⁹  
- 1 ≤ k ≤ 10⁹  
- `n` is even  

## Time Limit

1 second  

## Memory Limit

512 MB  

## Tags

arrays, sorting.
