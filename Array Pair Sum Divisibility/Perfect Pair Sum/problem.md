## Title

Perfect Pair Sum

## Slug

perfect-pair-sum

## Difficulty

Medium

## Description

You are given a list of integers.  
Your goal is to check if you can form pairs such that the `sum of each pair` is divisible by a given number `k`.

Return `"YES"` if this is possible, otherwise `"NO"`.

### Example 1

#### Input
4  
9 7 5 3  
6

#### Output
YES

### Example 2

#### Input
4  
92 75 65 48  
10

#### Output
YES

## Input Format

- First line: integer `n`.  
- Second line: array elements.  
- Third line: integer `k`.

## Output Format

- Print `"YES"` if pairing is possible, otherwise `"NO"`.

## Constraints

- 1 ≤ n ≤ 10⁵  
- 1 ≤ arr[i] ≤ 10⁹  
- 1 ≤ k ≤ 10⁹  
- `n` is even

## Time Limit

1 second

## Memory Limit

512 MB

## Tags

arrays, sorting.
