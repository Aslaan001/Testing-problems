## Title

Parity Pockets

## Slug

parity-pockets

## Difficulty

Medium

## Description

Count contiguous pockets in the given array where the pocket's sum equals its product. Output the total count.

## Examples

### 1

#### Input

4
1 2 3 2

#### Output
5

### 2

#### Input

3
2 2 2

#### Output
5

## Input Format

- First line: integer `n`.  
- Second line: `n` integers.

## Output Format

- A single integer — count of pockets where sum == product.

## Constraints

- 1 ≤ n ≤ 10³  
- 1 ≤ arr[i] ≤ 10⁴

## Time Limit

1 second

## Memory Limit

512 MB

arrays .
