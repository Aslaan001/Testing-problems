## Title

Balance Count

## Slug

balance-count

## Difficulty

Medium

## Description

Compute how many contiguous subarrays satisfy the property that their sum equals their product. Output that count.

## Examples

### 1

#### Input

4
1 2 3 2

#### Output
5

### 2

#### Input

3
2 2 2

#### Output
5

## Input Format

- First line: integer `n`.  
- Second line: `n` integers.

## Output Format

- Single integer — number of subarrays with sum == product.

## Constraints

- 1 ≤ n ≤ 10³  
- 1 ≤ arr[i] ≤ 10⁴

## Time Limit

1 second

## Memory Limit

512 MB

arrays .
