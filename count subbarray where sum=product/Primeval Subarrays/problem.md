## Title

Primeval Subarrays

## Slug

primeval-subarrays

## Difficulty

Medium

## Description

A sequence is ancient—some contiguous ranges satisfy sum == product. Count all such ranges (subarrays) and return the total.

## Examples

### 1

#### Input

4
1 2 3 2

#### Output
5

### 2

#### Input

3
2 2 2

#### Output
5

## Input Format

- First line: integer `n`.  
- Second line: `n` integers.

## Output Format

- A single integer — count of qualifying subarrays.

## Constraints

- 1 ≤ n ≤ 10³  
- 1 ≤ arr[i] ≤ 10⁴

## Time Limit

1 second

## Memory Limit

512 MB
