## Title

Mirror Subarrays

## Slug

mirror-subarrays

## Difficulty

Medium

## Description

A contiguous subarray is called a mirror if its sum equals its product. Count how many mirror subarrays exist in the given array.

## Examples

### 1

#### Input

4
1 2 3 2

#### Output
5

### 2

#### Input

3
2 2 2

#### Output
5

## Input Format

- First line: integer `n`.  
- Second line: `n` integers.

## Output Format

- One integer — count of mirror subarrays.

## Constraints

- 1 ≤ n ≤ 10³  
- 1 ≤ arr[i] ≤ 10⁴

## Time Limit

1 second

## Memory Limit

512 MB

arrays .
