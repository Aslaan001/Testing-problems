## Title

Seal of Balance

## Slug

seal-of-balance

## Difficulty

Medium

## Description

In the sequence of integers, identify the number of contiguous parts where the sum of values equals their product. Return the total count of such balanced seals.

## Examples

### 1

#### Input

4
1 2 3 2

#### Output
5

### 2

#### Input

3
2 2 2

#### Output
5

## Input Format

- First line: integer `n`.  
- Second line: `n` integers.

## Output Format

- A single integer — count of balanced subarrays.

## Constraints

- 1 ≤ n ≤ 10³  
- 1 ≤ arr[i] ≤ 10⁴

## Time Limit

1 second

## Memory Limit

512 MB

arrays .
