## Title
Festival Tent Donation Run

## Slug
festival-tent-donation-run

## Difficulty  
Medium

## Description

At a circular festival ground, donation tents are arranged in a ring. Collectors must gather donations but cannot visit two adjacent tents to avoid crowd disruption. Find the maximum donation collectible.

## Examples  

### 1  

#### Input  
3  
2 3 2  

#### Output  
3  


### 2  

#### Input  
4  
1 2 3 1  

#### Output  
4  


### 3  

#### Input  
3  
1 2 3  

#### Output  
3  


## Input Format  

- The first line contains an integer n — the number of houses.  
- The second line contains n space-separated integers vals[i].  


## Output Format  

Return a single integer — the maximum total amount that can be collected.


## Constraints  

- 1 ≤ n ≤ 100  
- 0 ≤ vals[i] ≤ 1000  


## Time Limit  
1 second  

## Memory Limit  
256 MB  


## Tags  
dynamic-programming, circular-array, optimization
